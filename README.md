カドポン村
==========

Multiplayer roguelike dungeon crawling board game with strong influences from Dokapon Kingdom and Path of Exile.

## Setup

Kadopon requires [NodeJS](https://nodejs.org/en/).

### Installation

```sh
$ npm install
```

### Tests

```sh
$ npm test
```
